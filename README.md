## Portofolio Proyek & Riset

### Geopolitical Supply Chain Risk Intelligence
Membangun aplikasi hibrida yang menggunakan DeepSeek API untuk mengekstrak data dari teks berita logistik, dan model XGBoost untuk memprediksi probabilitas gangguan rute beserta estimasi kerugian finansialnya.

### Churn Business Valuation
Merancang model regresi logistik untuk memprediksi probabilitas pelanggan berhenti berlangganan (*churn*) dan menghitung dampaknya terhadap valuasi bisnis komersial.

### Automated Data Storyteller
Mengembangkan aplikasi analitik yang mengubah data penjualan mentah (CSV) menjadi narasi bisnis dan rekomendasi strategis otomatis. Menggunakan Pandas untuk analisis statistik deskriptif, deteksi korelasi, serta tren temporal, lalu mengintegrasikan DeepSeek API untuk menghasilkan laporan naratif dalam Bahasa Indonesia. Antarmuka interaktif dibangun dengan Gradio dan di-deploy melalui Google Colab Pro.  

### Sistem Deteksi Anomali (Riset Asistensi)
